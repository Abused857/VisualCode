### Archivo `database.php`

Este archivo contiene la configuración para todas las conexiones de base de datos disponibles en tu aplicación Laravel. Aquí se especifica la conexión por defecto, así como los detalles específicos para cada tipo de base de datos soportada.

```
<?php

    | Default Database Connection Name
    |--------------------------------------------------------------------------
    |
    | Aquí puedes especificar cuál de las conexiones de base de datos deseas
    | usar por defecto para las operaciones de base de datos. Esta es la
    | conexión que se utilizará a menos que se especifique otra.
    |
    */

    'default' => env('DB_CONNECTION', 'sqlite'),

    /*
    |--------------------------------------------------------------------------
    | Database Connections
    |--------------------------------------------------------------------------
    |
    | Aquí están todas las conexiones de base de datos definidas para tu aplicación.
    | Se proporciona una configuración de ejemplo para cada sistema de base de datos
    | compatible con Laravel. Puedes agregar o eliminar conexiones según sea necesario.
    |
    */

    'connections' => [

        'sqlite' => [
            'driver' => 'sqlite',
            'url' => env('DB_URL'),
            'database' => env('DB_DATABASE', database_path('database.sqlite')),
            'prefix' => '',
            'foreign_key_constraints' => env('DB_FOREIGN_KEYS', true),
        ],

        'mysql' => [
            'driver' => 'mysql',
            'url' => env('DB_URL'),
            'host' => env('DB_HOST', '127.0.0.1'),
            'port' => env('DB_PORT', '3306'),

            'unix_socket' => env('DB_SOCKET', ''),
            'charset' => 'utf8mb4',
            'collation' => 'utf8mb4_unicode_ci',
            'prefix' => '',
            'prefix_indexes' => true,
            'strict' => false, // Cambiado a false para permitir consultas GROUP BY sin especificar todos los campos
            'engine' => null,
            'options' => extension_loaded('pdo_mysql') ? array_filter([
                PDO::MYSQL_ATTR_SSL_CA => env('MYSQL_ATTR_SSL_CA'),
                PDO::ATTR_PERSISTENT => true,
            ]) : [],

        'mariadb' => [
            'driver' => 'mariadb',

        ],

        'pgsql' => [
            'driver' => 'pgsql',
            'url' => env('DB_URL'),

            'port' => env('DB_PORT', '5432'),
            'database' => env('DB_DATABASE', 'laravel'),
            'username' => env('DB_USERNAME', 'root'),

            'charset' => 'utf8',
            'prefix' => '',
            'prefix_indexes' => true,
            'search_path' => 'public',
            'sslmode' => 'prefer',
        ],

        'sqlsrv' => [
            'driver' => 'sqlsrv',
            'url' => env('DB_URL'),
            'host' => env('DB_HOST', 'localhost'),
            'port' => env('DB_PORT', '1433'),
            'database' => env('DB_DATABASE', 'laravel'),
            'username' => env('DB_USERNAME', 'root'),
            'password' => env('DB_PASSWORD', ''),
            'charset' => 'utf8',
            'prefix' => '',
            'prefix_indexes' => true,

    ],
```

#### Explicación Adicional

- **`strict`**: Este parámetro se establece en `false` para las conexiones `mysql` y `mariadb`. Al configurarlo de esta manera, Laravel permite ejecutar consultas `GROUP BY` sin necesidad de especificar explícitamente todos los campos en la cláusula `GROUP BY`. Esto es útil cuando se utiliza MySQL en versiones que requieren configuraciones específicas para este tipo de consultas.
